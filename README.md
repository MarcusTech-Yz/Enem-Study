## Sobre o projeto

O **enem.study** nasceu como uma plataforma pessoal de estudos para o ENEM, como um estudante de ensino médio senti a necessidade de um sistema completo de integração de anotações, conteúdos relevantes e até mesmo plano completo de estudo semanal. 

A ideia principal é simples:

> O estudante não precisa apenas de uma lista de conteúdos.
> Ele precisa de uma rota que mostre o que estudar, por que estudar e como acompanhar sua evolução.

A plataforma combina plano de estudos, progresso, tópicos organizados, anotações, vídeos, questões, revisão e personalização de perfil em uma experiência única.

---

## Funcionalidades

### Plano de estudos inteligente

O sistema monta uma rota de estudos considerando:

* tempo disponível por dia;
* tamanho dos blocos de estudo;
* matérias prioritárias;
* tópicos ainda não estudados;
* domínio do estudante;
* revisões pendentes;
* equilíbrio entre áreas.

O plano pode ser recalculado automaticamente, mas também permite que o usuário monte sua própria rota de forma manual.

---

### Progresso com diagnóstico

A aba de progresso mostra mais do que porcentagem.

Ela acompanha:

* tempo total estudado;
* sequência de estudo;
* cobertura dos tópicos;
* domínio médio;
* concentração de tempo por matéria;
* pontos de atenção;
* próximos passos recomendados.

O objetivo é fazer o estudante entender se o estudo está realmente funcionando.

---

### Tópicos organizados por matéria

Cada matéria possui uma trilha de conteúdos baseada na estrutura do ENEM, com tópicos organizados para facilitar a navegação.

O estudante pode:

* buscar conteúdos rapidamente;
* abrir tópicos específicos;
* marcar tópicos como concluídos;
* registrar dificuldade;
* iniciar sessões de estudo;
* acompanhar o tempo estudado.

---

### Ambiente de estudo por tópico

Cada tópico possui uma área própria com ferramentas para estudar de forma ativa:

* anotações por blocos;
* vídeos salvos;
* questões manuais;
* revisões;
* exportação em Markdown;
* registro de dificuldade;
* sessão de foco com tempo estudado.

A ideia é transformar cada tópico em um pequeno caderno de estudo.

---

### Exportação Markdown

As anotações dos tópicos podem ser exportadas em `.md`, permitindo uso em ferramentas como:

* Obsidian;
* Notion;
* VS Code;
* editores Markdown em geral.

Também é possível copiar o Markdown ou importar uma anotação pronta.

---

### Perfil personalizável

O perfil funciona como um espaço de identidade do estudante dentro da plataforma.

Ele possui:

* nome personalizado;
* foto de perfil;
* wallpapers animados;
* molduras;
* nível;
* XP;
* sequência de estudo;
* matéria principal;
* estatísticas pessoais;
* conquistas.

A proposta é fazer o estudante sentir que o progresso dele tem uma identidade visual própria.

## Como rodar localmente

Clone o repositório:

```bash
git clone https://github.com/markussdev/Enem-Study.git
```

Entre na pasta do projeto:

```bash
cd Enem-Study
```

Abra o projeto em um servidor local.

Exemplo usando VS Code com a extensão Live Server:

```txt
Clique com o botão direito no index.html
Open with Live Server
```

## Objetivo

O **enem.study** é mais do que um planner.

Ele foi criado para ajudar estudantes a responderem três perguntas:

1. O que eu preciso estudar agora?
2. Por que esse conteúdo importa?
3. Meu estudo está realmente evoluindo?
